Understanding i18n Signals

What we look for in GitHub repositories

New Language File HIGH VALUE

A new translation file was added to the codebase (e.g., fr.json, es.yaml)

Meaning: The company is actively adding support for a new language - strong expansion signal.

i18n Pull Request HIGH VALUE

An open PR with localization keywords in the title

Meaning: Early signal of upcoming language support - PRs show intent before changes are merged.

Keyword Match LOW VALUE

Commits containing i18n-related keywords (translate, localization, etc.)

Meaning: General i18n activity - could be new features or maintenance. Worth investigating.
